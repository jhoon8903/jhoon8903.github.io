# React
### A JavaScript library for building user interfaces

##### 라이브러리 : 특정 프로그램에서 자주 사용되는 기능을 모아놓은 모음집
##### 사용자 인터페이스 (UI) : 버튼, 텍스트 입력창 등

- **REACT : Javascript Open Source UI Library**
