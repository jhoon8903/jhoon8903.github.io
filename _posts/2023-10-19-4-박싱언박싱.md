# Boxing
- object Type의 경우 `값` 형식의 데이터를 `heap에 할당`
```csharp
object a = 20;
```
![](https://i.imgur.com/aWt0xH2.jpg)

# UnBoxing
- 값 형식 데이터를 다시 객체에 할당 해야 하는 경우
```csharp
object a = 20;
int b = (int)a;
```
![](https://i.imgur.com/ab2Pz3c.jpg)

```csharp
private static void Boxing()  
{  
	int a = 123;  
	object b = a;  
	int c = (int)b;  
	  
	WriteLine($"Boxing: {b}");  
	WriteLine($"UnBoxing: {c}");  
	  
	double x = 3.1414213;  
	object y = x;  
	double z = (double)y;  
	  
	WriteLine($"Boxing: {y}");  
	WriteLine($"UnBoxing: {z}");  
}
```
